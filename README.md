# World Cup Oracle 🔮⚽

An open-source World Cup prediction simulator powered by statistics, chaos, and unreasonable national confidence.

> Predict the tournament. Blame the model later.

## Features

- Pick your national team
- Run Monte Carlo tournament simulations
- View champion probability rankings
- Check your team's path of hope
- Generate a shareable prediction card
- Argue with your friends using numbers that look serious

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Vercel
